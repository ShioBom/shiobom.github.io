清除浮动，元素水平垂直居中，文字显示省略号

元素水平垂直居中

  • 方式1：额外标签法

  • 方式2：
    position: absolute;

    top: 50%;
left: 50%;
margin-left: -50px;
margin-top: -50px;

  • 方式3
    position: absolute;

    top: 0;
left: 0;
right: 0;
bottom: 0;
margin: auto;

  • 方式4

清除浮动

  • 在浮动元素末尾添加一个空的标签(基本不使用)

  • 父元素设置 overflow：hidden

  • 父元素也设置浮动

    缺点：使得与父元素相邻的元素的布局会受到影响，不推荐使用

    • 使用:after 伪元素
单行文本溢出显示省略号
视口
  • width:可视区域的宽度，值可为数字或关键词device-width
  • intial-scale:页面首次被显示是可视区域的缩放级别，取值1.0则页面按实际尺寸显示，无任何缩放
  • maximum-scale=1.0, minimum-scale=1.0;
    可视区域的缩放级别 maximum-scale用户可将页面放大的程序，1.0将禁止用户放大到实际尺寸之上。
  • user-scalable:是否可对页面进行缩放，no 禁止缩放
